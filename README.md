# 🏎️ F1 Driver Performance Analysis

## 📌 Project Overview
This project analyzes Formula 1 race data to evaluate driver performance, consistency, and hidden insights beyond just race wins.

---

## 📊 Objectives
- Identify the most consistent drivers  
- Analyze driver performance using points and positions  
- Discover underrated (hidden) performers  

---

## 📂 Dataset
Dataset sourced from Kaggle Formula 1 dataset.

Files used:
- drivers.csv  
- results.csv  
- races.csv  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

---

## 📈 Analysis Performed
- Average finishing position  
- Total points scored  
- Consistency (standard deviation)  

---

## 🔍 Key Insights
- Consistency plays a major role in driver performance  
- Some drivers perform well despite fewer points  
- Hidden performers exist with strong average positions
